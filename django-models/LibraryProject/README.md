# LibraryProject

## Introduction

This project is part of the ALX Django Learning Lab.  
It introduces the basic setup of a Django development environment, including
project creation, running the development server, and understanding the default
Django project structure.

## Project Objective

- Install Django
- Create a Django project
- Run the Django development server
- Explore core Django files such as settings.py, urls.py, and manage.py

## How to Run the Project

1. Navigate into the project directory:
   ```bash
   cd LibraryProject
   ```
